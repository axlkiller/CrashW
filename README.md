## Installing :

```
apt update && apt upgrade -y
```
```
pkg install python3
```
```
pkg install git
```
```
git clone git://github.com/axlkiller/CrashW.git
```
```
cd CrashW
```
```
chmod +x Crashw.py
```


## Run :

Go to the CrashW Directoy by type `cd CrashW` and then type
```
python3 Crashw.py
```
